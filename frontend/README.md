# Frontend

Next.js App Router dashboard for the Multi-Agent Job Discovery & CV Advisor.

## Run

```bash
npm install
npm run dev
```

Set `NEXT_PUBLIC_API_BASE_URL=http://localhost:8000/api` when the backend is not on the default URL.

## Pages

- `/login`
- `/dashboard`
- `/upload-cv`
- `/job-sources`
- `/jobs`
- `/match`
- `/skill-gap`
- `/roadmap`
- `/cv-improve`
- `/interview`
- `/agent-trace`
